  3. JAVA源文件是由什么组成的？一个源文件必须要有public 类吗？
 
 - java 是面对对象 ，JAVA应用程序的源文件是由若干个书写形式相互独立的类组成的
 
 - 如果源文件中有许多个类，那么只能有一个类是 public 类，如果有一个类是public类，那么源文件的名字必须与这个类名字相同
