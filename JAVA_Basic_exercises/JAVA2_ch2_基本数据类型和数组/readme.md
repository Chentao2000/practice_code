小结：
 1. 标识符是由字母、下划线、美元符号和数组组成的，并且第一个字符不能是数字字符
 2. JAVA 语言有8中基本数据类型 :boolean 、byte、short、int 、long、float、char
 3. 数组是相同类型的数据元素安顺序组成的一种复合数据类型，数组属于引用型变量 ，因此两个相同类型的数组如果具有相同的引用，他们就有相同的元素
