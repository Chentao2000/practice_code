c++ Delete: 用法：

Delete是一个运算符，用于销毁由新表达式创建的数组和非数组(指针)对象。
- 可以通过使用Delete运算符或Delete []运算符来使用Delete
- New运算符用于动态内存分配，从而将变量放在堆内存中。
- 这意味着Delete运算符从堆中释放内存。
- 指向对象的指针不会被破坏，指针所指向的值或存储块将被破坏。
